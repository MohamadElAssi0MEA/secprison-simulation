# SecuPrison Simulation Engine
A hybrid OSI-layer vulnerability simulator that converts abstract vulnerabilities into prison-security analogies.

## Features
- One vulnerability per OSI layer
- Prison analogy visual mapping
- Checklist generation (Read-Do or Read-Confirm)
- Node CLI + React UI

## Run Simulation
```
npm install
node src/index.js
```

## Run Tests
```
npm test
```

## Run UI
```
cd ui
npm install
npm run dev
```
